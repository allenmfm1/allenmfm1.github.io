GANXIANG

感想
通过学习这次夏令营，我收获颇多。从一名学习机械的学生，开始接触计算机知识，我感到我的机会已经来了，我认为技多不压身，多学一门技术，以后就多一个出路，多一个选择。我会更加努力，让自己更加优秀。随着信息技术的高速发展并迅速渗透到社会生活的各个方面，计算机日益成为人们学习，工作生活必不可少的基本工具。
开学一个学期了，从刚刚步入大学对于计算机什么也不了解大现在上了几节C语言，计算机导论课，课堂中布置的任务还是挺多的，毕竟打字也慢理解也慢，很多时候打代码都像是热锅上的蚂蚁,什么符号什么字母都找不到，什么都不知道还要不停地问如何操作，这也许算是积累的知识和经验吧。
在老师的建议下我们用着博客，看着自己的打字速度在提升这也是一种能力吧
学习计算机要坚持多做多学多看，自己多尝试。老师说一开始学计算机就是从模仿来的，学习照这给好的多练，不是说模仿不好，只有在学习中才能慢慢找到适合的方式找到属于自己的路。
在学习中我也发现了要想好好的了解只能靠自己去亲身体会，去实践，把各项命令的键记熟，积极地去自己动手操作，提高自己的操作能力，锻炼自己。
我也在学习中明白计算机的更新有多迅速，随着教育体制的改革和教育理念的更新以及信息技术的飞速发展技术新的教育知识，改变传统的教育观念充实我们的专业技能，我们必须不断学习掌握最新的知识，才能把工作做得更好。在学习中我也发现了自己和同学间存在很大的差距，感觉一开始就落人一步，但是在未来对于计算机的学习中我也会更加努力通过与老师的交流积累经验，提示自己的技术能力。
此次夏令营，我受益匪浅，我会继续学习更加努力。

11号
Gridea搭建博客网站，此网站和qq和微信一样，可以安装在电脑上。 Onenote是一个云计算笔记应用。
在访问Gridea网站时，最好使用谷歌浏览器
第一步：搜索网站
第二步：
登录 coding.net【此网站为github的国内版，因为github为外国网站，此网站是中国的】
点击右上角【个人版登陆】—-【马上注册】—-【qq登录】—-【微信扫码关联】—-【完成注册】
进入页面之后点击右上角【小加号】—-【项目】—-【填写项目名称】—-【填写用户名.coding.me】{用户名需点开右上角头像个人设置，然后查看，可以更改，但只可改一次}

10号 把网站放到github网站上
【自己的地址：xuanxuan0829.github.io】 【 https://github.com/xuanxuan0829/xuanxuan0829.github.io.git 】
【以后这个就是赵仪宣个人网站的网址了】

sudo npm install hexo-deployer-git --save  需要在blog下执行 

双击桌面文件目录—进去打开html文件
在type后输入：【一个空格】git 
在下一行输入：repository：【一个空格】https://github.com/xuanxuan0829/xuanxuan0829.github.io.git
最后一行：branch:【一个空格】master
点关闭然后保存

然后执行下列代码：
{
cd blog
sudo npm install hexo-deployer-git –save
git config –global user.email “2538428177@qq.com“
git config –global user.name “xuanxuan0829”
gedit _config.yml
hexo g
hexo d
}
更新网页内容：
cd blog 【如果是在blog格式下，则不用这个代码，如果不是，则使用此行代码】
hexo new “TITLE”
gedit ~/…md 【在这个步骤后会弹出一个修改内容的页面，修改然后保存即可】
hexo g 【此步骤为编译】
hexo d 【此步骤为发表】
删除网页文章：
cd blog
rm -f hexo “TITLE”
hexo g
hexo d
