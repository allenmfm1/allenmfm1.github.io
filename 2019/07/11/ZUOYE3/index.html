第三天 9号
在虚拟机中打开浏览器：输入localhost:4000 【打开的页面就是刚搭建好的博客】
{提示：在linux系统中命令绝对不能少空格，但是可以多空格}

如何更新网站中的内容：
{
hexo new “在此输入新的文章标题”
gedit 复制上条命令电脑出来的代码{但是要安装，输入：
文本框中的绿色三条杠不要动，在下面编辑内容，编辑完成之后点关闭，然后保存。
编辑完成不代表发表
发表：hexo g [此步骤为转换为网页形式]
发布：hexo d [发布]
hexo s
}
具体更新代码：
【1】 hexo new “TITLE”
【2】 gedit ~/…md
【3】 hexo g
【4】 hexo d
【5】 hexo s
