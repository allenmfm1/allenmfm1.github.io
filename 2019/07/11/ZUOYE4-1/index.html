10号 把网站放到github网站上
【自己的地址：xuanxuan0829.github.io】 【 https://github.com/xuanxuan0829/xuanxuan0829.github.io.git 】
【以后这个就是赵仪宣个人网站的网址了】

sudo npm install hexo-deployer-git --save  需要在blog下执行 

双击桌面文件目录—进去打开html文件
在type后输入：【一个空格】git 
在下一行输入：repository：【一个空格】https://github.com/xuanxuan0829/xuanxuan0829.github.io.git
最后一行：branch:【一个空格】master
点关闭然后保存

然后执行下列代码：
{
cd blog
sudo npm install hexo-deployer-git –save
git config –global user.email “2538428177@qq.com“
git config –global user.name “xuanxuan0829”
gedit _config.yml
hexo g
hexo d
}
更新网页内容：
cd blog 【如果是在blog格式下，则不用这个代码，如果不是，则使用此行代码】
hexo new “TITLE”
gedit ~/…md 【在这个步骤后会弹出一个修改内容的页面，修改然后保存即可】
hexo g 【此步骤为编译】
hexo d 【此步骤为发表】
删除网页文章：
cd blog
rm -f hexo “TITLE”
hexo g
hexo d
